# Pipe Sizing Calculator

This Python application calculates the appropriate pipe diameter for a given flow rate, velocity, and fluid properties, ensuring safe and efficient process flow design.

## Features
- Input flowrate (m³/h, L/min, kg/h)
- Select acceptable velocity range (10-30 m/s recommended)
- Output minimum pipe diameter
- Warns if velocity is too high or low

## Why it matters
Correct pipe sizing ensures:
- Reduced pressure drop
- Lower pumping costs
- Longer equipment life
